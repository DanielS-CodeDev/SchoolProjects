This project was created in CSC345 (Analysis of Discrete Structures). This project I had to pick three unique and efficient sorting algo. 
which sorts the input file lines into three categories and then prints out how much damage Logic (main character) takes from Lord Bug's
attacks from the input file while following other rules set by the spec.

Designed a program that parses an input file to group data into 3 distinct arrays that are then sorted so they could be sorted independently.  
Evaluated and selected 3 unique sorting algorithms based on data types, array utilization and applying Big-O asymptotic analysis to optimize performance. 
Implemented Counting Sort for String arrays, Merge Sort for stable integer sorting and Quicksort with a median-of-three pivot selection to prevent the O(n^2) worst case.
