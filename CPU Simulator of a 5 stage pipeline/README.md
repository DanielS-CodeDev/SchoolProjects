This class project for CSC252 (Computer Organization) was created to help me better understand how the 5 stage pipeline works and how forwarding works as well.

- Simulated pipeline stages including Instruction Fetch, Instruction Decode, Execute, Memory and Write-back utilizing simulated ALU multiplexers (MUXes) and branch/ execution logic
- Incorporated hazards/stalls detection logic for load and store word instructions to model how data hazards are identified and handled within a 5 stage pipeline
