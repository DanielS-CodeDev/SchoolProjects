This project was created in CSC345 (Analysis of Discrete Structures) this project I had first create and
adjacency list from the input file then find the MCST then perform Dijkstra's Algo. to find the shorted
path from SR to BL which includes vertex and edge weights. The main file is 'labyrinth.java'.

- Engineered a program that parses an input file to construct an adjacency list optimizing memory efficiency by avoiding matrix overhead.
- Evaluated and implemented Kruskal’s algorithm over Prims to determine the Minimum Cost Spanning Tree (MCST) from the adjacency list due to custom tie-breaking logic the project specified. 
- Executed Dijkstra’s Algorithm on the graph from the adjacency list to calculate the shortest path from source and destination nodes factoring in edge and vertex weights.
