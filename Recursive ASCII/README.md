This project was created in CSC120 (Intro to Computer Programming 2). This project I was limited to only code everything 
with recursive functions and based on input of a single letter.  'p' a park, 'e' an empty lot and 'b' for a building to be
drawn. This project was to focus on just recursion.

- Developed a program that can generate procedural ASCII art of a street based on custom user configuration input. 
- Designed the entire program under the strict restraint of only using recursive logic to code this program.
