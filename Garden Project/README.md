This project was created in CSC210 (Software Development). This project I had to first create a UML diagram and plan out the project then had 
to implement the UML diagram. This project also focused on inheritance and polymorphism.  The main file is 'RunCarden.java'.

- Designed a comprehensive Unified Modeling Language (UML) diagram to design program layout, methods and pre-validating code logic to eliminate unworkable paths in development. 
- Implemented the UML diagram in Java utilizing inheritance and polymorphism across Plant sub classes to build a scalable and modular simulation.
