This project was created in CSC345 (Analysis of Discrete Structures). This project I had to create an orthogonal list and based on the input file Logic (main character)
teleports to the coord. and searches in a cross pattern with certain restrictions on it. The main file is 'search.java'.

Constructed an orthogonal list to map and manage coordinates within a simulated 100x100 grid system parsed from file input.
Incorporated a custom cross-directional search pattern constrained by a simulated time limit and a search window to traverse the grid and then output the elapsed game time.
